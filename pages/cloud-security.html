Interview Q&A

Q: Explain the shared responsibility model.

"The cloud provider and customer share security responsibilities. What changes by service model: In IaaS, you manage everything from OS up. In PaaS, provider handles OS/runtime. In SaaS, provider handles everything except data and access. Key point: You always own your data and identity controls regardless of model."

Q: What's the difference between CSPM and CWPP?

"CSPM (Posture) looks at configuration - are your cloud resources securely configured? Storage buckets public? MFA enabled? CWPP (Workload) protects what's running inside - anti-malware, vulnerability scanning, runtime protection. Think: CSPM = outside the box (config), CWPP = inside the box (workload)."

Q: What are the top cloud security risks?

"Top risks: 1) Misconfiguration - public storage, open security groups. 2) Excessive permissions - overprivileged accounts, no MFA. 3) Lack of visibility - shadow IT, no logging. 4) Insecure APIs - exposed keys, no auth. 5) Data exposure - unencrypted data, poor access controls."
