Manufacturing & Distribution Infrastructure

Coca-Cola Bottling operates 50+ manufacturing plants and 200+ distribution centers across North America. Each plant has OT (Operational Technology) networks running bottling lines, while distribution centers focus on warehouse management and logistics. All locations connect to Azure via SD-WAN or MPLS.

Key Interview Points

Plant Architecture:

  • • Full Purdue Model implementation
  • • 3 firewall pairs per plant (IT, iDMZ, OT)
  • • Claroty for OT visibility
  • • Historian in iDMZ for data transfer
  • • Jump server for remote access

Distribution Center:

  • • Simpler architecture (no OT)
  • • Single edge firewall
  • • Full IT security stack (MDE, Intune)
  • • SD-WAN with dual transport
  • • WMS system on isolated VLAN
