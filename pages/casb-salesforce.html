Salesforce CASB Policies

Mass Download Prevention

Active

Block report exports exceeding 10,000 records from unmanaged devices.

Type: Session Policy
Action: Block + Alert
Scope: All Users
Severity: High

PII/PCI Data Protection

Active

Detect and quarantine files containing credit card numbers or SSN in Salesforce attachments.

Type: File Policy
Action: Quarantine
SITs: CC, SSN, SIN
Severity: Critical

Impossible Travel Detection

Active

Detect logins from geographically impossible locations within short time windows.

Type: Anomaly Policy
Action: Alert + Require MFA
Scope: All Users
Severity: High

Risky OAuth App Control

Active

Block third-party apps with excessive permissions from connecting to Salesforce.

Type: App Governance
Action: Revoke + Block
Permissions: High-risk
Severity: High

Sentinel Integration Query

Hunt for Salesforce security events from Defender for Cloud Apps in Microsoft Sentinel.

// Salesforce CASB Events from Defender for Cloud Apps
// Data source: CloudAppEvents table

CloudAppEvents
| where TimeGenerated > ago(24h)
| where Application == "Salesforce"
| extend 
    EventCategory = case(
        ActionType has "Download", "Data Exfiltration",
        ActionType has "Login", "Authentication",
        ActionType has "Permission", "Privilege Change",
        ActionType has "Delete", "Data Destruction",
        "Other"
    ),
    RiskLevel = case(
        IsAnomaly == true, "High",
        ActionType has_any ("MassDownload", "ImpossibleTravel"), "Critical",
        "Medium"
    )
| project
    TimeGenerated,
    AccountDisplayName,
    ActionType,
    EventCategory,
    RiskLevel,
    IPAddress,
    CountryCode,
    City,
    IsAnomaly,
    ActivityObjects
| order by TimeGenerated desc
