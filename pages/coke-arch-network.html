Network Security Appliances in Azure

Coca-Cola's Azure environment uses a layered network security approach with multiple firewall types and load balancers. Azure Firewall handles east-west and basic north-south traffic, while Palo Alto NVAs provide advanced inspection for sensitive workloads. Application Gateway with WAF protects web applications.

Key Interview Points

Firewall Strategy:

  • • Azure FW: General traffic, TLS inspection
  • • Palo Alto: SAP/sensitive workloads
  • • Both zone-redundant for HA
  • • Centralized policy (FW Policy, Panorama)
  • • All traffic logged to Sentinel

Load Balancer Design:

  • • Front Door: Global, DDoS, CDN
  • • App Gateway: Regional L7, WAF
  • • Standard LB: HA for VMs/NVAs
  • • ILB for internal services (SAP, SQL)
  • • HA Ports for cluster failover
