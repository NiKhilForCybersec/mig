The 14 Tactics (Attack Chain)

PRE-ATTACK PHASE
TA0043

Reconnaissance

Gather info about target. OSINT, scanning, social media.

TA0042

Resource Development

Build attack infrastructure. Buy domains, create accounts.

INITIAL COMPROMISE
TA0001

Initial Access

Get into the network. Phishing (T1566), exploit public apps, valid accounts, supply chain.

ESTABLISH FOOTHOLD
TA0002

Execution

Run malicious code. PowerShell (T1059.001), cmd, scripting.

TA0003

Persistence

Stay in the network. Scheduled tasks (T1053), registry keys, services.

ESCALATE & EXPAND
TA0004

Privilege Escalation

Get higher permissions. Exploits, misconfigs, token manipulation.

TA0005

Defense Evasion

Avoid detection. Disable AV (T1562), obfuscation, living off land.

TA0006

Credential Access

Steal credentials. Mimikatz (T1003), keylogging, brute force.

TA0007

Discovery

Learn the environment. Network scanning, account enumeration.

TA0008

Lateral Movement

Move through network. RDP (T1021.001), SMB, PsExec, WMI.

MISSION EXECUTION
TA0009

Collection

Gather target data. Screen capture, keylogging, clipboard.

TA0011

Command & Control

Maintain communication. HTTPS C2 (T1071), DNS tunneling.

TA0010

Exfiltration

Steal data out. C2 channel, cloud storage, physical media.

TA0040

Impact

Disrupt/destroy. Ransomware (T1486), wiper, defacement.

Interview Q&A

Q: What is MITRE ATT&CK and how do you use it?

"MITRE ATT&CK is a knowledge base of adversary tactics, techniques, and procedures based on real-world observations. It has 14 tactics (the WHY - like Initial Access, Persistence) and 200+ techniques (the HOW - like Phishing, PowerShell). I use it for: Detection engineering - mapping our rules to techniques to find coverage gaps. Threat hunting - hunting for specific techniques in our environment. Incident response - understanding attacker behavior during investigations. It's the universal language of cyber threats."

Q: Walk me through an attack using ATT&CK terminology.

"A ransomware attack: Initial Access via phishing email (T1566) with malicious attachment. Execution using PowerShell (T1059.001). Persistence through scheduled task (T1053). Defense Evasion by disabling Windows Defender (T1562). Credential Access using Mimikatz to dump LSASS (T1003.001). Lateral Movement via RDP (T1021.001) to other systems. Impact - data encryption for ransom (T1486). Each technique has specific detections - this mapping helps us ensure coverage."

Q: How do you identify gaps in your detection coverage?

"Map existing detection rules to ATT&CK techniques - this creates a coverage matrix. Use tools like ATT&CK Navigator to visualize gaps. Prioritize based on: 1) Techniques used by threat actors targeting your industry, 2) Techniques with high impact (credential access, lateral movement), 3) Techniques you have data sources for. For gaps, either build new detections or accept risk with compensating controls. Regular reviews as new techniques emerge."
